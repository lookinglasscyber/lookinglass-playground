# Lookinglass Reference Library

This folder stores reusable templates and reference material for Lookinglass Cyber LLC.

## Template Catalog

- **Statements of Work (SOWs):** Scope, deliverables, and acceptance language for each service tier.
- **Proposals:** Client-facing summaries with pricing, assumptions, and win themes.
- **Service Catalogs:** Menu of offerings with value props, target client profiles, and SLA details.
- **Onboarding Kits:** Step-by-step runbooks, checklists, and kickoff questionnaires.
- **Reporting Packs:** Sample assessments, posture reports, and cadence review decks.
- **Support Docs:** FAQ sheets, escalation paths, and tooling configuration guides.

Update this README whenever new template categories get added so the workspace stays organized.
